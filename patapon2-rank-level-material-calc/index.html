計算式
・必要素材数 = S(ランク, 目標レベル) − S(ランク, 現在レベル)
　・S(r, L) = r * q * (q + 1) / 2 + m * (q + 1)
　　・q = floor(L / r)
　　・m = L % r
・必要チャリン = base × ( (現在レベル+1)+…+目標レベル )
メモ
  • 素材: 1〜4（素材3はLv3から、素材4はLv6から要求開始）
  • ランク: 1〜5
  • チャリン: 10〜60
  • 現在レベル: 0〜10
  • 目標レベル: 1〜10
  • 「計算」で、現在→目標の差分（必要数）を算出します。
